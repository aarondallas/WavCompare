##########
WavCompare
##########

Compare two WAVE files using Fast Fourier Transform

Synopsis
========

.. code-block:: python

    from wavcompare import FFT
    if FFT.wavs_match(wav1, wav2):
        print("WAV files seem to be the same)


Dependencies
============

`Wav2Numpy <https://github.com/aarondallas/Wav2NumPy>`_
